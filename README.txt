input 0 - 2 - тесты для проверки работы всех типов матриц поотдельности
input 3 - тест для проверки работы всех типов вместе
input 4 - тест для проверки работы рандомной генерации матриц
input 5-6 - тест с некорректными входными данными для рандоной генерации
input 7-8 - некорректные входные данные для введных вручную матриц
для запуска тестов введите input<number>.txt в Run->Edit configuration->Parameters
выходные данные при корректных входных будут лежать в файле output.txt
сообщение об ошибке из-за некорректных данных выводятся в консоль
